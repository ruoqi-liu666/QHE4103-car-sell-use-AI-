# QHE4103-car-sell-use-AI-

Seller module prototype for the QHE4103 Online Car Sale project.

Pages
- `register.html`: Seller registration with client-side validation
- `login.html`: Seller login
- `add-car.html`: Add car listing form with live preview

Shared assets
- `styles.css`: Shared seller-module visual system, real car photography, and responsive layout
- `script.js`: Validation, localStorage demo data, seller-side interactions, and add-car live preview
